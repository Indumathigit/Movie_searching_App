# MovieZone

A movie search app built with ReactJS and OMDB API.

## Features

- Search movies by title
- Filter by type (Movie, Series, Episode)
- Pagination for search results
- Click any movie to see full details
- Movie detail page with poster, plot, cast, ratings
- Error handling for no results

## Tech Stack

- ReactJS
- React Router
- Vite
- Tailwind CSS
- OMDB API

## How to Run

1. Clone the repo
2. Install packages
   npm install
3. Start the app
   npm run dev
4. Open browser at http://localhost:5173

## API Used

OMDB API - https://www.omdbapi.com/

## Pages

- Home Page (/) - Search movies with filter and pagination
- Detail Page (/movie/:id) - Full movie details
